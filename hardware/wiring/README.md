# Arduino Micro 手焊接线

## 结构

13 个按键采用 4×4 矩阵，其中 3 个位置空着。每个按键串联一个 1N4148 二极管，方向统一使用 QMK 的 `COL2ROW` 约定。Rev 0.4 使用一个带按压开关的 EC11，预留给未来音量控制。

QMK 手焊指南对矩阵、二极管和 Pro Micro 控制板有完整说明：<https://docs.qmk.fm/hand_wire>。

## 连接原则

- 按键矩阵的 4 行连接 Arduino Micro 的 D4～D7。
- 按键矩阵的 4 列连接 D8～D11。
- 矩阵二极管的黑色环方向必须统一。
- 音量 EC11 的 A/B/按压分别连接 D2/D3/D12。
- 12 颗独立 quota RGB 灯的 DIN 接 A3，VCC 接 5V，GND 接 GND；DIN 方向从控制器侧进入第 0 颗灯。
- 12 颗灯沿键盘面板窗口逐颗排列，左侧为第 0 灯，右侧为第 11 灯；也可使用裁切后的 12 像素 RGB 灯带。
- 在 DIN 串联约 330 Ω 电阻，并在灯带电源入口并联 1000 µF 以上电解电容。

## 手焊安全

- 焊接、改线和测量电阻时拔掉 USB。
- 第一次上电只连接 Arduino Micro，不连接 RGB。
- 不要把 12 颗 RGB 灯全白满亮度直接从 USB 长时间供电；首版建议亮度不超过 20～30%。
- 先用万用表确认 5V 与 GND 没有短路。
- USB 插口要在外壳上做支撑，避免 Micro-USB 焊盘受力。
