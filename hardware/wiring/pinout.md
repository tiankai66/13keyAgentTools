# Rev 0.7 引脚表

以下是 Arduino Micro 的 Arduino 引脚命名。`A0`～`A3` 既可以作模拟输入，也可以作为数字输入/输出；本项目按具体功能使用。

| 功能 | Arduino 引脚 | 方向 | 说明 |
|---|---:|---|---|
| Matrix row 0 | D4 | 输出 | 逐行扫描 |
| Matrix row 1 | D5 | 输出 | 逐行扫描 |
| Matrix row 2 | D6 | 输出 | 逐行扫描 |
| Matrix row 3 | D7 | 输出 | 逐行扫描 |
| Matrix col 0 | D8 | 输入上拉 | 读取按键 |
| Matrix col 1 | D9 | 输入上拉 | 读取按键 |
| Matrix col 2 | D10 | 输入上拉 | 读取按键 |
| Matrix col 3 | D11 | 输入上拉 | 读取按键 |
| Volume encoder A | D2 | 输入上拉 | 音量旋转 |
| Volume encoder B | D3 | 输入上拉 | 音量旋转 |
| Volume encoder push | D12 | 输入上拉 | 音量旋钮按下 |
| Quota RGB pixel DIN | A3 | 输出 | 12 颗 WS2812B / SK6812 独立灯珠串联数据 |

## Rev 0.7 两层布局

```text
[左上 1 键] [12 颗水平面板 RGB quota 灯][音量旋钮]
[        4 × 3 按键矩阵        ]
[上层控制面板，下面用支柱连接下层主板面板]
[下层主板面板：右侧红框下方的 Arduino Micro 承托架 + 图示前侧 USB-C 开口]
```

音量旋钮使用 D2/D3/D12。面板上的 12 颗 RGB 灯使用 A3 串联，按灯珠顺序从左到右显示剩余用量；12 颗灯共用一个数据引脚，通过 DIN/DOUT 串联，不占用 12 个 GPIO。

Arduino Micro 固定在右侧红框下方的独立下层主板面板上，主板不再放在按键矩阵中部；上层面板增加透明主板观察窗，USB-C 从图示前侧上边缘进入。建议先将板卡和 USB 线固定，再把上层控制面板通过四角 M3 支柱合上，最后焊矩阵与 RGB 灯链。

## 矩阵位置

电气矩阵中的 4×4 位置如下，`--` 是未安装的电气位置；面板物理布局与此一致：

```text
[ A ][ --][ --][ --]
[ B ][ C ][ D ][ E ]
[ F ][ G ][ H ][ I ]
[ J ][ K ][ L ][ M ]
```

如果后续改成 2U 键帽，先保留两个独立开关，等固件确认后再把它们映射为同一个动作。
