# 迭代记录

## 2026-07-18 — 0.1.0

### 决策

- 新建独立工程 `13keyAgentTools`，不修改同级的旧 `project/` 目录。
- 第一版采用 Arduino Micro，而不是直接设计专用 PCB。
- 13 个按键采用 4×4 矩阵，控制设备占用独立引脚。
- 第一版使用 USB 有线连接，不引入蓝牙、电池和充电电路。

### 下一步

- 打印 `mechanical/cad/agent_macro.scad` 的 `tolerance` 部件。
- 采购并测试 6 个按键、1 个 EC11、1 个 Arduino Micro。
- 在没有 RGB 的情况下完成矩阵和旋钮输入验证。

## 2026-07-18 — 0.2.0 设计迭代

### 决策

- 将外壳加宽为 184 × 112 mm，在按键区和控制区之间加入低矮中心指托与凸起定位点。
- 增加第二个 EC11，独立预留给系统音量；当前固件先用 F11/F12 占位。
- 前侧加入 12 灯 WS2812B/SK6812 灯带窗口，用 `QUOTA 0..100` 显示当前 Agent 剩余用量。
- 灯带亮度限制为 32/255，优先保证 USB 供电和键盘输入稳定。

### 下一步

- 打印 Rev 0.2 公差件和外壳，验证中心指托高度、旋钮间距及灯带窗口。
- 采购两个 EC11 和一段 12 灯 5V 灯带，完成 D2/D3/D12 与 A3 的手焊。
